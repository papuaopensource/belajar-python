# Contributing to Belajar-Python

Terima kasih telah berminat berkontribusi pada **Belajar-Python**.
Proyek ini terbuka untuk siapa pun yang ingin membantu pengembangan baik dari segi pengembangan ***project***, materi, dokumentasi dan lain-lain untuk dampak yang lebih baik bagi semua orang.

## Cara Berkontribusi

1. **Fork repositori** ini ke akun GitHub Anda.
2. Buat **branch baru** untuk setiap perubahan, fitur, atau perbaikan.
3. Pastikan perubahan Anda **jelas**, tidak merusak fitur yang sudah ada, dan sesuai dengan tujuan proyek.
4. Ajukan **pull request** dengan penjelasan singkat mengenai perubahan yang dilakukan.
5. Jika perubahan cukup besar, sertakan penjelasan tambahan atau catatan teknis yang diperlukan.

## Struktur Proyek

```sh
├── public/          # File statis
├── src/
│   ├── assets/      # Aset (gambar, dll)
│   ├── components/  # Komponen UI & playground
│   ├── content/     # Konten blog
│   ├── layouts/     # Layout halaman
│   ├── lib/         # Utilitas (Pyodide, dll)
│   ├── pages/       # Halaman & materi belajar
│   ├── styles/      # CSS
│   └── i18n/        # Dukungan multi-bahasa
├── astro.config.mjs # Konfigurasi Astro
├── tailwind.config.mjs # Konfigurasi Tailwind
├── tsconfig.json    # Konfigurasi TypeScript
└── package.json    # Dependensi & script
```

## Pedoman Umum

* Jaga struktur kode tetap rapi dan mudah dipahami.
* Usahakan mengikuti pola atau gaya penulisan kode yang sudah ada.
* Diskusikan terlebih dahulu melalui *issue* jika ingin mengajukan fitur baru yang signifikan.
* Berikan deskripsi yang jelas pada setiap *commit*.

Selengkapnya baca di [CODE OF CONDUCT](./CODE_OF_CONDUCT.md)

## Dokumentasi Lengkap

Dokumentasi kontribusi yang lebih lengkap, termasuk standar kode, alur pengembangan, dan panduan teknis lainnya, akan ditambahkan segera.
